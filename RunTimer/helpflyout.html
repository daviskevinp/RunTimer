How to use RunTimer
RunTimer is an application that allows you to track your pace as you run a selected distance in a given time. You adjust your current speed to match what you're running on the treadmill, and the application calculates the speed at which you'll need to run in order to make your time goal
In this application demo, the distance is fixed at 3.1 miles (5 km), and the target time is 31 minutes. Future versions will make these numbers adjustable from a settings menu.
